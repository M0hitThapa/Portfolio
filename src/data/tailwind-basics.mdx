### Next.js Setup

```bash

npx tailwindcss init -p
```

Add file paths in `tailwind.config.js`:

```js
module.exports = {
  content: [
    "./pages/**/*.{js,ts,jsx,tsx}",
    "./components/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
};
```

Include Tailwind styles:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```
